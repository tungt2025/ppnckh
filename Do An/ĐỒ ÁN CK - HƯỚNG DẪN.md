Bước đầu tiên là phải chọn chủ đề (research topics) -- bài toán (research problem).

Chủ đề là liên quan đến computer vision, pattern recognition, machine learning.

Bài toán có thể chia làm 2 dạng: bài toán tự nghĩ ra - xuất phát từ nhu cầu thực tế, và bài toán do người khác đã nghĩ ra. Các bạn nên chọn dạng thứ 2.

Để chọn bài toán, hãy vào xem các bài báo trong các hội nghị hàng đầu như CVPR gần nhất (trong vòng 5 năm trở lại), chọn ra một số bài toán mà mình cảm thấy hứng thú (mỗi bài báo trong hội nghị hàng đầu này giải quyết một bài toán nào đó).

Với mỗi bài toán, tìm hiểu sơ lược về input, output, algorithms/proposals/techniques (để chuyển từ input sang output). Tìm hiểu các bộ dữ liệu dùng trong bài báo (tại sao cần bộ dữ liệu?), tìm hiểu về source code của bài báo.

Bài báo được chọn sẽ gồm các tiêu chí sau:
- Phù hợp với sở thích
- Có PUBLIC dataset
- Có source code (trên github)
- Có demo minh hoạ trên YouTube

Nhóm trưởng đại diện mỗi nhóm điền các thông tin cơ bản sau để tiện trao đổi:
- Tên bài báo, tên tác giả, tên hội nghị, năm
- Link bài báo, link YouTube, link Github
- Paste Abstract và Conclusion của bài báo
- Intput/Output theo mô tả của bài báo
- Kết quả của bài báo
